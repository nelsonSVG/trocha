# Plan: Adapt Agarrando Trocha to Monteadentro.cc Visual System

## 1. Research Summary: Monteadentro.cc Design System

### Typography
- **Primary Font**: Montserrat (Google Fonts)
- **Weights Loaded**: 100, 200, 300, 400, 500, 600, 700, 800, 900 (normal + italic)
- **Usage Patterns**:
  - Headings: Montserrat 600-700 weight, uppercase, letter-spacing -0.9px
  - Body text: Montserrat 400-500 weight
  - Navigation: Montserrat uppercase, tracking
  - Footer title: Montserrat 600, 30px, uppercase, letter-spacing -0.9px

### Color Palette
- **Background**: #000000 (black), #ffffff (white)
- **Text**: #ffffff (on dark), #000000 (on light)
- **Accent/CTA**: White text on black backgrounds
- **Borders/Dividers**: rgba(0,0,0,0.1), rgba(255,255,255,0.1)
- **Social icons**: White on black backgrounds

### Layout & Spacing
- **Container**: Boxed layout (max-width ~1140px)
- **Section padding**: Generous vertical spacing (99px spacers observed)
- **Grid**: Elementor-based responsive grid
- **Breakpoints**: Mobile 767px, Tablet 1024px, Desktop 1440px

### Header Design
- **Style**: Transparent background, becomes sticky on scroll (offset: 500px)
- **Logo**: Text-based logo image (black on transparent)
- **Navigation**: Horizontal menu, uppercase, spaced links
- **Social**: Instagram icon in header
- **CTA Button**: Contact button (hidden on mobile/tablet/desktop - popup form)

### Footer Design
- **Background**: #000000 (solid black)
- **Logo**: White circular logo centered
- **Title**: "Monteadentro" in white, Montserrat 600, 30px, uppercase
- **Subtitle**: "BIKEPACKING BLOG" smaller text
- **Social Icons**: Instagram, Facebook - white on black
- **Credit**: "Rew" logo/link
- **Layout**: Centered, single column

### Animations
- **Scroll animations**: fadeInDown, fadeInLeft, fadeInRight, fadeInUp, zoomIn
- **Hover effects**: Shrink, grow, pulse, pop, grow-rotate, wobble-skew
- **Transitions**: 0.3s for backgrounds, borders, border-radius, box-shadow
- **Smooth scrolling**: Enabled

### UI Components
- **Buttons**: Rounded, uppercase text, border-based with hover fills
- **Cards**: Image overlays, hover effects
- **Forms**: Clean fields with placeholders, submit button
- **Social Icons**: Rounded shape, centered alignment

## 5. Key Design Decisions

### Font Mapping (CONFIRMED: Full Montserrat)
| Current | New | Reason |
|---------|-----|--------|
| Playfair Display (display) | Montserrat 700 | Bold headings |
| Lora (serif) | Montserrat 400 | Body text |
| Inter (sans) | Montserrat 400 | UI elements |
| DM Sans (accent) | Montserrat 500 | Accents |
| JetBrains Mono (mono) | Montserrat 400 | Monospace not needed |

**Decision**: Replace ALL 5 fonts with Montserrat only, matching Monteadentro.cc exactly.

### Color Adjustments
- Keep black/white dominant scheme (already similar)
- Ensure proper contrast ratios
- Use rgba for subtle borders and overlays

### Spacing Philosophy
- Generous vertical padding (80-120px between sections)
- Centered content with max-width containers
- Consistent letter-spacing for uppercase text
