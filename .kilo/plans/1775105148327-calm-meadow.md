# Plan: Favicon, OG Image y Optimización SEO

## Resumen
Configurar favicon circular con fondo negro, crear OG Image optimizado para redes sociales, y actualizar metadata SEO basada en investigación del proyecto Agarrando Trocha y su público objetivo.

## 1. Crear Nuevo Favicon SVG

### Descripción
Crear un favicon SVG optimizado (32x32px) con el símbolo AT blanco sobre fondo circular negro.

### Archivo a crear/modificar
- `public/favicon.svg` (reemplazar)

### Especificaciones técnicas
- Tamaño: viewBox="0 0 32 32"
- Fondo: Círculo negro (#000000) centrado
- Logo: Símbolo AT blanco extraído del SVG original
- Fill blanco: #FFFFFF en todos los paths
- Escalado proporcional manteniendo la forma del logo

### Pasos
1. Extraer los paths del archivo `documents and material/Logo/SVG/AT WHITE 1.svg`
2. Crear SVG con círculo negro de fondo (cx="16", cy="16", r="16")
3. Insertar el símbolo AT con transformación de escala adecuada (aprox scale(0.055))
4. Asegurar que todos los paths tengan fill="#fff"
5. Centrar el logo en el círculo

## 2. Crear Nuevo OG Image SVG

### Descripción
Crear OG Image (1200x630px) con el logo AT blanco sobre fondo negro completo para compartir en redes sociales.

### Archivo a crear/modificar
- `public/og-image.svg` (reemplazar)

### Especificaciones técnicas
- Dimensiones: 1200x630px (aspect ratio 1.91:1 - estándar OG)
- Fondo: Rectángulo negro completo (#000000)
- Logo: Símbolo AT blanco centrado
- Posición: Centrado tanto horizontal como verticalmente
- Margen: Espacio adecuado alrededor del logo para que respire

### Pasos
1. Crear SVG con viewBox="0 0 1200 630"
2. Rectángulo de fondo negro (#000000)
3. Extraer y posicionar el símbolo AT centrado
4. Aplicar escala proporcional para que el logo ocupe aproximadamente 60-70% del ancho
5. Asegurar fill="#fff" en todos los paths
